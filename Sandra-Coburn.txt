
    1. What is Semantic HTML? 
The use of HTML markup to provide meaning to the content in a webpage, helpful for users and developers as well as browsers.

    2. What is HTML used for? 
To tell the browser how to interpret and read the information so the page can be viewable.

    3. What is an attribute and where do we put it? 
It provides additional information about an element. You put it right at the start tag.

    4. What is the h1 tag used for? How many times should I use it on a page?
The h1 tag is used to accentuate the main heading title of a page. It's better to use it once but if necessary more times to show more than one relevant point a page.
